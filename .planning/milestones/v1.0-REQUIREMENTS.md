## Complete Requirements

### Install (INST)

- [x] **INST-01** — `install.sh` detects Arch vs Artix at runtime via `/etc/os-release`
- [x] **INST-02** — Correct PPD package variant per distro
- [x] **INST-03** — Enable `power-profiles-daemon` service via correct init
- [x] **INST-04** — Fails loudly with actionable error on unsupported distros
- [x] **INST-05** — Idempotent: re-running on configured system does not fail
- [x] **INST-06** — Only installs C utilities from `utils/`; does not copy `scripts/`
- [x] **INST-07** — Refuses to run as root (UID=0)
- [x] **INST-08** — Detects chroot / live-ISO environments and warns
- [x] **INST-09** — Ships udev rule (`90-backlight.rules`)
- [x] **INST-10** — Adds installing user to `video` and `input` groups
- [x] **INST-11** — Prints post-install verification summary
- [x] **INST-12** — Runtime deps updated: add `brightnessctl`, drop `xorg-xbacklight`

### Migration (MIG)

- [x] **MIG-03** — Documentation drift fixed: `CLAUDE.md` references betterlockscreen and powerprofilesctl; Void Linux references removed
- [x] **MIG-04** — Repo hygiene: remove `dwm/p1.rej`; fix `.gitignore` typo

### Battery (BATT)

- [x] **BATT-01** — `battery-notify` auto-detects battery via glob, exits 0 on desktops
- [x] **BATT-02** — Urgency-tiered notifications: low (20%) → normal, critical (5%) → critical (persistent), different replace-ids
- [x] **BATT-03** — Thresholds configurable via `config.def.h`
- [x] **BATT-04** — Tier state machine with on-disk state file, fires only on transitions
- [x] **BATT-05** — Schmitt hysteresis: fires low at ≤20%, re-arms at ≥25%
- [x] **BATT-06** — Invoked from `dwm-start` on 30s supervisor loop

### Brightness (BRIGHT)

- [x] **BRIGHT-01** — New C utility invokes `brightnessctl -c backlight`, emits dunst notification
- [x] **BRIGHT-02** — Wired into `dwm/config.h` for brightness keys
- [x] **BRIGHT-03** — `brightnessctl` added to runtime deps (covered by INST-12)
- [x] **BRIGHT-04** — Minimum brightness floor 5%, step size 5%, flock serialization
- [x] **BRIGHT-05** — Uses dunst replace-id 9101, progress-bar hint

### Release (REL)

- [x] **REL-01** — README overhaul: project purpose, supported distros, install instructions, keybinding reference, troubleshooting
- [x] **REL-02** — Screenshots in README showing dwm bar, dmenu, dunst notifications
- [x] **REL-03** — VM verification documentation
